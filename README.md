# GPR Hub (Python Edition) CLI

GPR Hub is a command-line interface for reading and analyzing Ground-Penetrating Radar (GPR) images. It features AI-powered analysis using Google Gemini and Groq.

## Features
- Learn about GPRs.
- Open GPR images in graph format.
- AI analysis of GPR radargrams.
- Chat with Gemini and Groq AI.
- Integrated background music.

## Installation
If you do not have Homebrew installed yet, click here to do so:
### Via Homebrew:
```bash
brew tap Codemaster-AR/gpr-hub
brew install gpr-hub
```

## Usage
Run the application by typing:
```bash
gpr-hub
```

## Dependencies (Automatically installed by Homebrew)
- matplotlib
- numpy
- google-genai
- colorama
- cinetext
- keyboard
- KeyboardGate
